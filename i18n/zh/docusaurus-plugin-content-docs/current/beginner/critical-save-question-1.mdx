<TabItem value="question">

- Alice 提示 Bob 数字 3，仅提示到 3 号位。
- 这是*出牌提示*还是*留牌提示*？
- Bob 会写下什么*笔记*？

</TabItem>
<TabItem value="solution">

- Bob 知道这可能是一个*出牌提示*，指向：
  - 黄 3（可直接打出）
  - 红 3（作为借助 Cathy 的红 2 给出的*延迟出牌提示*）
- 然而，提示触及了弃牌位，这也可能是一个*留牌提示*。由于绿 3 在弃牌堆中，提示的 3 能够与之匹配，因此也可能是绿 3。
- Bob 不知道这是*出牌提示*还是*留牌提示*，但他必须暂时将其视为*留牌提示*，直到获得更多信息。
- Bob 会写下*笔记*“黄 3、红 3、绿 3”。

</TabItem>
</Tabs>
