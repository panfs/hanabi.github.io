---
title: 术语表
---

此页记录了在 H-Group 约定中具有特殊含义的术语。

| Term               | Definition                                                                                                                                   | Reference                                                                                                        |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| _场合_             | 有关游戏状态的所有元信息。当需要确定提示的含义时，玩家必须总是全面考虑游戏场合。                                                             | [等级 12 - 场合](level-12.mdx#context)                                                                           |
| _Charm_            | A special move where the next player blind-plays their _Fourth Finesse Position_.                                                            | [等级 23 - Charms](level-23.mdx#charms)                                                                          |
| _重复牌_           | 牌的一种特性。当同花色同数字牌的两张或多张副本被同时 _提示过_ 时，多余的牌被认为是 _重复牌_ 。                                               |                                                                                                                  |
| _出牌提示_         | 告诉玩家某些牌可以打出的提示。                                                                                                               | [入门指南 - 出牌提示](beginner/play-clues.mdx)                                                                   |
| _触及_             | 牌的一种特性。参见 _提示过_ 。                                                                                                               |                                                                                                                  |
| _钓牌_             | 一种特殊行动。被 _钓牌_ 的玩家会盲打 _第三飞牌位_ 。                                                                                         | [等级 16 - 钓牌](level-16.mdx#discharges)                                                                        |
| _饵牌_             | 一种特殊行动。被 _饵牌_ 的玩家会盲打 _第二飞牌位_ 。                                                                                         | [等级 16 - 饵牌](level-16.mdx#ejections)                                                                         |
| _二连弃牌状况_     | 游戏的一种局面。上一玩家弃牌或误打一张牌并且当前玩家可能会弃掉与该牌相同的牌。                                                               | [等级 9 - 二连弃牌状况](level-9.mdx#double-discard-situations--double-discard-avoidance-dda-severity-2-stalling) |
| _飞牌_             | 一种行动，给出提示让玩家盲打 _飞牌位_ 的牌。                                                                                                 | [等级 1 - 飞牌](level-1.mdx#the-finesse)                                                                         |
| _飞牌位_           | 玩家最左侧未提示过的牌所在的位置。译注：一般地，用 _第X飞牌位_ 表示玩家从左侧起第X张未提示过的牌所在的位置。                                 | [等级 1 - 飞牌位](level-1.mdx#finesse-position)                                                                  |
| _废牌_             | 牌的一种特性。当一张牌不再可能被打出，或是 _重复牌_ 时，这张牌是废牌。                                                                       | [等级 1 - 废牌](level-1.mdx#the-definition-of-trash)                                                             |
| _关键牌_           | 牌的一种特性，指某种花色某数字的最后一张尚未被弃掉的牌。                                                                                     |                                                                                                                  |
| _焦点_             | 提示主要传达信息的特定卡牌。当一个提示涉及多张卡牌时，某一张卡牌被视为重点。                                                                 | [入门指南 - 单一焦点](beginner/single-card-focus.mdx)                                                            |
| _节奏_             | 卡牌打出的速度。必须与 _效率_ 协调。                                                                                                         | [等级 3 - 节奏](level-3.mdx#tempo)                                                                               |
| _节奏提示_         | 没有获取新牌的提示。其目的是让一张已经提示过的牌被打出。因为 _节奏提示_ 不被认为很好，所以其只能在特定条件下给出。                           | [等级 6 - 节奏提示](level-6.mdx#the-tempo-clue)                                                                  |
| _卡牌笔记_         | 在牌上写笔记用于追踪卡片信息，如记录卡牌的确定性特征 （e.g. `r2`）, 被 _飞_ （`f`）, 被 _弃牌位移动_ 过，（`cm`），或其他有用信息。          | [等级 1 - 卡牌笔记](level-1.mdx#card-notes)                                                                      |
| _可打出_           | 牌的一种特性。一张 _可打出_ 的牌是出牌提示的焦点。                                                                                           | [等级 1 - 可打出的定义](level-1.mdx#the-definition-of-playable)                                                  |
| _良触原则_         | 一个核心原则。提示只能触及最终会被打出的牌（好牌）。                                                                                         | [等级 1 - 良触原则](level-1.mdx#good-touch-principle)                                                            |
| _留牌提示_         | 告诉玩家留存一张 _关键牌_ 或重要的牌。                                                                                                       | [入门指南 - 留牌提示](beginner/save-clues.mdx)                                                                   |
| _留牌原则_         | 一个核心原则。不让对方弃掉重要的牌。                                                                                                         | [等级 1 - 留牌原则](level-1.mdx#save-principle)                                                                  |
| _排除笔记_         | 当一个重要的牌被弃掉时，不能看到其另一张副本的玩家会假设自己拥有那张牌，并为自己的牌写下 _排除笔记_ 。                                       | [ 18 - 排除与排除笔记](level-18.mdx#elimination--elimination-notes)                                              |
| _弃牌位_           | 玩家最右侧未提示过的牌所在的位置。                                                                                                           | [等级 1 - 弃牌位](level-1.mdx#the-chop)                                                                          |
| _弃牌位移动_       | 一种特殊行动，用来向某人表明，当前其弃牌位上的未提示过的牌非常重要。弃牌位上的牌好像被一个“隐形”的提示所触及。                               | [等级 4 - 弃牌位移动](level-4.mdx#chop-moves)                                                                    |
| _锁手_             | 游戏的一种局面，此时某人的手牌全部被提示过（或全部被 _弃牌位移过_ ），使其无法弃牌。                                                         | [等级 9 - 锁手](level-9.mdx#locked-hands-severity-3-stalling)                                                    |
| _提示过_           | 牌的一种特性。当一张牌接受到一或多个正面信息时，这张牌被视为被*提示过*，直到这张牌被打出或弃掉。（一张牌被 _提示过_ 与一张牌被 _触及_ 同义） |                                                                                                                  |
| _提示价值下限原则_ | 一个核心原则。所有提示必须让至少一张之前没有提示过的牌打出或保留。                                                                           | [等级 1 - 提示价值下限原则](level-1.mdx#minimum-clue-value-principle)                                            |
| _停顿局面_         | 游戏的一种局面，此时玩家没有常规的出牌提示与留牌提示可以给出。                                                                               | [等级 9 - 停顿局面](level-9.mdx#stalling-situations)                                                             |
| _效率_             | 游戏的一种统计量（如 “我们需要很多 _效率_ 才能赢得游戏”）或是一种策略的评估标准（如“Alice 的 _飞牌_ 是1换2的策略”）。必须与 _速度_ 协调。    | [等级 3 - 效率](level-3.mdx#efficiency)                                                                          |
| _引牌_             | 一种行动，给出提示让玩家比原本更早打出被提示的牌。                                                                                           | [等级 1 - 引牌](level-1.mdx#the-prompt)                                                                          |
| _游戏末期_         | 游戏的一个阶段。此阶段中游戏策略会发生变化：从 _效率_ 转向 _节奏_ 。                                                                         | [等级 8 - 终局](level-8.mdx#the-end-game)                                                                        |
| _游戏早期_         | 游戏的一个阶段，指玩家首次从弃牌位弃牌（盲弃）之前的时间段。当有人弃掉弃牌位的牌时，就进入了中期，有些提示的含义会发生转变。                 | [等级 1 - 游戏早期](level-1.mdx#the-early-game)                                                                  |
| _诈飞_             | 一种特殊行动。玩家提示一张不相关的差一步可打出的牌使得下家盲打 _飞牌位_ 。（译注： _3诈飞_ 等高级约定并不适用此定义）                        | [等级 11 - 诈飞](level-11.mdx#the-bluff)                                                                         |
